# Preview Assets

Store exported preview assets for template cards, detail pages, and future demo surfaces in this directory.

## Conventions

- Use `images/` for thumbnail previews, for example `assets/previews/images/ldv-beamer.png`.
- Use `documents/` for preview PDFs, for example `assets/previews/documents/ldv-beamer.pdf`.
- Keep thumbnails lightweight and web-friendly.
- If a template has no preview thumbnail yet, keep `preview_image = null`.
- If a template has no preview PDF yet, keep `preview_document = null`.
